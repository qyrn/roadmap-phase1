## 📋 Phase 1 — Foundations (Feb–Apr 2026)

8-week sprint. Platforms: TryHackMe, OverTheWire, Root Me.

| Week | Focus | Key deliverables |
|------|-------|-----------------|
| S1 (Feb 18–22) | Linux Fundamentals | THM Linux Fund. 1/2/3, Bandit 0–15, blog article |
| S2 (Feb 23 – Mar 1) | Linux Advanced + Bash | Bash scripting, Root Me App Script, Bandit 15–25, 3 scripts |
| S3 (Mar 2–8) | Network Fundamentals | THM Networking + Wireshark, cheatsheet networking.md |
| S4 (Mar 9–15) | Nmap & Recon | THM Nmap + PrivEsc, Shodan/crt.sh, cheatsheet nmap.md |
| S5 (Mar 16–22) | Python for Pentest | Port Scanner (CV project), published on GitHub |
| S6 (Mar 23–29) | Burp Suite & HTTP | THM Burp + OWASP Top 10, PortSwigger labs, cheatsheet burp.md |
| S7 (Mar 30 – Apr 5) | Buffer + OSINT Project | OSINT tool (CV project): WHOIS, subdomains, DNS |
| S8 (Apr 6–12) | Web Vulns Intro | THM SQLi, Root Me Web, PortSwigger Access Control / Auth |

**Phase 1 gate:** Quiz ≥70% + Port Scanner + OSINT tool published + ≥3 blog articles.
